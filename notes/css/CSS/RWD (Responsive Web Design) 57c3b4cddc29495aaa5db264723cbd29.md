# RWD 原則

---

- 能用 flexbox 就用 flexbox 處理
- 不能用 flexbox 就用 `media query` 處理
    
    ```css
    /* example */
    @media screen and (max-width: 612px){
      nav ul{
        flex-direction: column;
      }
    }
    ```
