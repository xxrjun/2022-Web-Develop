### 2. 網路 internet

任兩台裝置相連，即形成網路

### 3. `https` 協議

早期資料傳輸是基於 http 協議，但因為其傳輸方式的過程都是透過明文傳輸，使得資料若是被竊盜，竊盜者就可以輕鬆讀取傳輸資料。因此現今多數網站都是基於 https 協議進行資料傳輸，後面多的 s 是 secure 的意思。https 透過 http 進行資料傳輸，但通訊過程使用 SSL/TSL 加密，使其資料傳輸獲得較多安全性上的保障。
